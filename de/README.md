# Easy Scripts

**von Otis C.**

## Über das Repository

In diesem Repository findet ihr alle meine „Easy Scripts“.
Die Idee hinter den Skripten ist, meine täglichen Linux-Tools deutlich einfacher und intuitiver zu gestalten.
Deshalb möchte ich sie mit euch allen teilen.

## Skriptindex und Anwendungsbeispiel

- [easy-gpg.sh](https://github.com/olc161/easy-scripts/blob/main/en/easy-gpg.sh)
  - easy-gpg ist mein Skript zum Erstellen, Löschen und Verwalten meiner GPG-Schlüssel in der Shell. Ich wollte mich nicht mit den einzelnen Befehlen auseinandersetzen müssen. Ich wollte einfach ein CLI-Tool, das alles für mich erledigt.

- [easy-shred.sh](https://github.com/olc161/easy-scripts/blob/main/en/easy-shred.sh)
  - easy-shred ist ein einfaches Skript, das den grundlegenden Befehl „shred“ vereinfacht, sodass Sie sich nicht mit der Syntax auseinandersetzen müssen.

- [easy-sync.sh](https://github.com/olc161/easy-scripts/blob/main/en/easy-sync.sh)
  - easy-sync ist ein einfaches CLI-Tool zum Verschieben oder Kopieren von Daten zwischen Geräten oder Servern.

- [easy-startup.sh](https://github.com/olc161/easy-scripts/blob/main/en/easy-startup.sh)
  - easy-startup ist mein kleines Startprogramm, das bei jedem Hochfahren meines PCs ausgeführt wird, um die wichtigsten Programme zu starten.
